本儀表板使用 GitHub Actions 每日自動抓取數據。
請按照以下步驟完成初次設定：

① 在 GitHub Repo 設定 Secret
Settings → Secrets and variables → Actions → New repository secret
名稱：FRED_API_KEY　值：你的 FRED API Key

② 手動執行一次 GitHub Actions
Actions → 「📊 Update Macro Dashboard Data」→ Run workflow

③ 等待 data.json 生成後刷新頁面
約 2–3 分鐘後，數據就會自動出現。之後每個工作日 14:00（台灣）自動更新。
